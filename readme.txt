== Description ==

Makes Twenty Fourteen's Ephemera Widget available in any other theme.

== Installation ==

1. Upload `ephemera-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
